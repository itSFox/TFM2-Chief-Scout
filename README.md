# Chief Scout: Best Player Finder

Chief Scout is a native mod for **Teamfight Manager 2** that helps you find the best players in your current save.

It analyzes players, free agents, transfer targets, role fit, potential, stats, upgrade value, and recommendation tags.

## Steam Workshop

Recommended installation method:

[Subscribe on Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3741344550)

Use manual GitHub releases only if you need a specific mod build for a specific Teamfight Manager 2 game version.

## Compatibility

| Teamfight Manager 2 version | Chief Scout version | Status |
|---|---:|---|
| 0.4.12 rollback | [v1.0.4](https://github.com/itSFox/TFM2-Chief-Scout/releases/tag/v1.0.4) | Stable |
| 0.4.13+ | Waiting for game-side UI fix | Known UI text regression |

## Manual releases

Archived manual builds are available in [GitHub Releases](https://github.com/itSFox/TFM2-Chief-Scout/releases).

Each release is named by both mod version and supported game version.

Example:

`ChiefScout_v1.0.4_for_TFM2_0.4.12.zip`

## Manual installation

1. Download the correct release archive for your game version.
2. Open the Teamfight Manager 2 game folder.
3. Open or create the `mods` folder.
4. Extract the `scout_assistant` folder into `mods`.

Final path should look like this:

`Teamfight Manager 2/mods/scout_assistant/mod.mod_info`

Do not rename the `scout_assistant` folder.
